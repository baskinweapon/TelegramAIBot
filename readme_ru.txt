Этот код является Телеграм ботом, реализованным с использованием библиотеки python telegram-bot.

Функционал:

Бот имеет два режима работы: один с использованием базы данных, другой с использованием языковой модели GPT от OpenAI.
В режиме работы с базой данных, бот получает сообщения от пользователя, обрабатывает их и возвращает лучший ответ из базы данных.
В режиме OpenAI, бот взаимодействует с пользователем с помощью языковой модели GPT для более природной конференции.
Бот имеет возможность собирать информацию от пользователя.
Бот имеет возможность "лайкать" или "дизлайкать" ответы и соответственно настраивать вес ответа в базе данных.
Требования:

telegram
telegram-ext
Использование:

Замените токен в файле Scripts/Token.py своим Telegram Bot API токеном.
в файле Token.py необходимо вставить свой токен:
    от бота телеграмма в переменную - token
    от chatGPT - openai_api_key

в файле Data.json храниться база данных для ответов,
 что-бы добавить новую информацию используйте шаблон:

   {
    "weight": 30,
    "value": "получить паспорт & загранпаспорт & не выдают загранпаспорт",
    "answer": "Вам не требуется приносить никаких справок из военкомата для оформления загранпаспорта. МВД самостоятельно запрашивает всю информацию. Отказать в выдаче загранпаспорта могут только если человек призван на военную службу или направлен на альтернативную гражданскую службу, - до окончания военной службы или альтернативной гражданской службы;",
    "tags": [
      "#war",
      "#mobilize"
    ],
    "need_question": false
   },

в файле main.py есть функция "def collect(message):" эта функция вызывавется после сбора вопросов
в этой функции нужно добавить отправку данных в ваш колл центр

в файле Data.py хранитятся дежурные ответы

у бота есть два режима работы:
    1. работа с базой данных и отправка пользователю ответов
    2. работа chatGPT (эта функция эксперементальная)
